Sistema de Gestão da Esquadra

Gestão de ocorrências policiais com árvores e estruturas de dados

Visão Geral

Aplicação desktop desenvolvida em Java para gestão de ocorrências policiais numa esquadra. O sistema utiliza estruturas de dados avançadas, com foco em árvores binárias, permitindo registo, pesquisa, organização e análise eficiente das ocorrências.

Funcionalidades

Registo de Ocorrências

Inserção estruturada de ocorrências policiais.

Árvore Binária

Armazenamento eficiente com ordenação e pesquisa rápida.

Pesquisa Avançada

Consulta por critérios como ID, tipo ou prioridade.

Menu Interativo

Interface em consola organizada e fácil de utilizar.

Implementação Técnica

Estrutura em Árvore

As ocorrências são armazenadas numa árvore binária, permitindo inserção, pesquisa e travessias eficientes.

Separação de Responsabilidades

Organização clara entre classes de dados, estruturas e menu de interação.

Validação de Dados

Controlo de entradas inválidas para garantir integridade da informação.

Projeto Académico
