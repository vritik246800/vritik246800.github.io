Node Note

Gestor de notas com interface moderna para macOS

Visão Geral

Aplicação nativa de notas desenvolvida em SwiftUI para macOS, com foco em produtividade e design minimalista estilo glass. As notas são organizadas numa estrutura de nós — cada nota é um nó independente, permitindo navegação rápida e filtragem por categoria. Persistência garantida com CoreData, sem dependência de cloud ou serviços externos.

Funcionalidades

Criar & Editar Notas

Editor inline com suporte a texto rico. Criação instantânea com atalho de teclado.

Organização por Nós

Estrutura em nós navegáveis — cada nota tem título, corpo e categoria atribuída.

Pesquisa Rápida

Filtro em tempo real por título ou conteúdo usando binding com @State.

Persistência CoreData

Todas as notas são guardadas localmente com CoreData — sem perda de dados entre sessões.

UI Glass macOS

Interface com Material nativo do SwiftUI — sidebar translúcida e painéis em glass.

Sidebar Navegável

Layout de três colunas — categorias, lista de notas e editor — típico de apps nativas macOS.

Stack Técnica

Swift SwiftUI macOS CoreData Combine MVVM

Implementação Técnica

Layout NavigationSplitView

Usa NavigationSplitView do SwiftUI para o layout de três colunas — padrão nativo macOS com sidebar colapsável e estado persistente entre sessões.

Reatividade com Combine

Pesquisa em tempo real implementada com @Published e debounce do Combine — filtra sem travar a UI mesmo em listas grandes.

CoreData + FetchRequest

Notas carregadas com @FetchRequest diretamente nas Views, com predicados dinâmicos para filtrar por categoria ou texto de pesquisa.

Glass Morphism Nativo

Sidebar e painéis usam .background(.ultraThinMaterial) para integração visual com o sistema macOS — sem assets ou libs externas.

Estado & Evolução

Atual: App funcional com criação, edição, organização por categorias, pesquisa e persistência completa.

Possíveis evoluções: sincronização com iCloud via NSPersistentCloudKitContainer, exportação de notas em Markdown, suporte a tags múltiplas por nota e modo de foco fullscreen.
