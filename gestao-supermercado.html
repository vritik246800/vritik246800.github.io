Sistema de Gestão de Supermercado

Gestão de produtos, clientes, vendas e faturação

Visão Geral

Aplicação desktop desenvolvida em Java para gestão completa de um supermercado ou mercearia. O sistema permite o controlo de produtos, clientes, vendas, cálculo automático de preços com IVA, geração de recibos e persistência de dados.

Funcionalidades

Gestão de Produtos

Registo, edição e cálculo automático do preço final.

Gestão de Clientes

Criação e consulta de clientes com histórico de compras.

Vendas & Faturação

Geração de recibos com IVA e margem de lucro.

Exportação em PDF

Emissão de faturas prontas para impressão.

Stack Técnica

Java Swing AWT POO I/O Files PDF

Arquitetura

Modelo

Produto.java Cliente.java Venda.java

Interface

Menu.java GestaoProdutos.java GestaoClientes.java

Persistência

FileManager.java PDFGenerator.java

Implementação Técnica

Cálculo Automático

O sistema calcula automaticamente o preço final com IVA e margem de lucro configurável.

Tabelas Interativas

Utilização de JTable para visualização e edição direta dos produtos.

Persistência Local

Dados guardados em ficheiros para garantir continuidade entre execuções.

Faturação Profissional

Geração de PDFs com data, hora e código único de fatura.

Estado & Evolução

Atual: Sistema completo e funcional para gestão de supermercados.

Evoluções futuras: base de dados, backup automático, autenticação de utilizadores e integração com impressoras POS.
