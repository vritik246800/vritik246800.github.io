Remote Access (LAN)

Acesso remoto seguro com Java, sockets e criptografia

Visão Geral

Aplicação desktop desenvolvida em Java que permite acesso remoto entre computadores através de uma rede local (LAN). O sistema implementa captura de ecrã em tempo real, comunicação cliente-servidor, criptografia AES e interface gráfica interativa.

Funcionalidades

Acesso Remoto em Tempo Real

Captura e transmissão contínua do ecrã via sockets TCP.

Comunicação Cliente-Servidor

Arquitetura dedicada com separação clara de responsabilidades.

Criptografia AES

Dados transmitidos de forma segura usando cifra simétrica.

Compressão de Imagem

Redução do tamanho das imagens para melhor desempenho.

Interface Gráfica

UI em Swing com logs, estado do servidor e controlos visuais.

Execução em LAN

Deteção automática de IP local e ligação direta entre máquinas.

Stack Técnica

Java Swing Sockets TCP/IP Multithreading AES Encryption Image I/O

Arquitetura do Projeto

Client

Server

Suporte

Implementação Técnica

Captura de Ecrã

Utilização da classe Robot para capturar o ecrã do servidor em intervalos regulares (~30 FPS).

Transmissão Segura

As imagens são enviadas através de streams criptografadas usando AES para proteger os dados.

Multithreading

Threads dedicadas garantem fluidez na comunicação e atualização contínua do ecrã remoto.

Interface Interativa

Interface Swing com logs em tempo real, indicadores de estado e controlo do servidor/cliente.

Estado & Evolução

Estado atual: Sistema funcional para acesso remoto em rede local com transmissão segura.

Evoluções futuras: controlo de teclado e rato, autenticação avançada, suporte WAN e otimização de largura de banda.
