Mensagens LAN

Comunicação em tempo real na rede local

⬇ Download Project

Visão Geral

Aplicação desktop em Java para troca de mensagens entre dispositivos na mesma rede local. Deteta automaticamente utilizadores online e permite comunicação instantânea via sockets TCP.

Funcionalidades

Deteção Automática

Identifica dispositivos ativos na rede sem configuração manual

Chat em Tempo Real

Troca instantânea de mensagens de texto

Histórico Local

Armazenamento de conversas no dispositivo

Interface Swing

GUI nativa e responsiva

Stack Técnica

Java Swing/AWT Sockets TCP Threads Redes LAN

Arquitetura

Network

Interface

ChatWindow.java UserListPanel.java

Utilitários

MessageUtils.java

Demonstração

Implementação Técnica

Descoberta de Dispositivos

Broadcast UDP para detetar peers na sub-rede. Cada cliente responde com hostname e IP.

Comunicação

Sockets TCP para envio/receção de mensagens. Thread dedicada por conexão ativa.

Interface

Swing com painel de utilizadores, área de chat e input field. Event-driven architecture.

Estado & Evolução

Atual: Sistema funcional para mensagens de texto em LAN.

Próximos passos: Transferência de ficheiros, encriptação E2E, mensagens de áudio.
