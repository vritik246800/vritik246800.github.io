Transferência de Ficheiros LAN

Envio de ficheiros entre dispositivos na rede local

Visão Geral

Aplicação desktop em Java para transferência direta de ficheiros entre dispositivos na mesma rede local. Identifica peers automaticamente e permite envio de ficheiros ou diretórios completos via sockets TCP.

Funcionalidades

Deteção de Dispositivos

Scan automático da sub-rede para encontrar peers disponíveis

Envio de Ficheiros

Transferência de ficheiros individuais ou diretórios completos

Barra de Progresso

Acompanhamento visual da transferência em tempo real

Confirmação de Receção

Dialog de aceitação/rejeição antes de receber ficheiros

Stack Técnica

Implementação Técnica

Descoberta de Peers

Ping na sub-rede (ex: 192.168.1.0/24) para identificar dispositivos com o serviço ativo na porta específica.

Protocolo de Transferência

Socket TCP envia metadata (nome, tamanho) seguido dos bytes do ficheiro. Recetor valida e confirma receção.

Gestão de Threads

Thread separada para scan de rede, envio e receção. SwingWorker para atualizar UI durante transferência.

Buffer de I/O

BufferedInputStream/OutputStream com chunks de 8KB para transferência eficiente de ficheiros grandes.

Estado & Evolução

Atual: Sistema funcional para transferência de ficheiros em LAN.

Próximos passos: Encriptação AES, compressão ZIP, autenticação de dispositivos, retry automático em falhas.
